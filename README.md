
# 🎬 Movie Data Analysis Dashboard

> A professional, interactive **data analytics dashboard** built using Streamlit to explore and visualize movie datasets with actionable insights.

---


## 📖 Project Description

The **Movie Data Analysis Dashboard** is a data-driven web application that enables users to analyze movie datasets through intuitive visualizations and structured insights.

This project demonstrates practical skills in:
- Data Analysis
- Data Visualization
- Dashboard Development
- Python-based Web Apps

---

## ✨ Features

✔️ Secure Login Authentication  
✔️ Interactive Sidebar Navigation  
✔️ Full Dataset Exploration  
✔️ Color vs Black & White Movie Analysis  
✔️ Genre Extraction & Insights  
✔️ Country-wise Movie Distribution  
✔️ Top 5 High-Budget Movies  
✔️ Year-based Filtering (2009 Movies)  
✔️ Custom Dataset Upload Support  

---

| Module | Description |
